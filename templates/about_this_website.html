About

Welcome to the VTU 6th Semester Fullstack Development Lab Documentation

This documentation website provides a walkthrough on how to execute and run the VTU 6th semester Fullstack Development Lab programs. The main idea of this documentation is to provide a comprehensive guide for running the Lab programs, "Sometimes you gotta run before you can walk ~Iron Man -Stan lee" If we get the program running we can understand how the things work the way they are and later we can tweak and modify according to our needs.

What's in this website

  • I've witten detailed walkthrough on how to run the programs and also provided video summary on how to run the program at the end of each page.
  • You can also download all the programs here
  • Additionally, I have also provided a link to the official Django documentation and other documentation on each page of programs where you can find detailed information about the methods and techniques used in the lab programs.
  • If you spot any mistakes, feel free to inform me while listening to this song.
